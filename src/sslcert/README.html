How to install the root certificate

In order to make the browser trust the https certificate for localhost, you should add the root.crt certificate. This is how to do it in different browsers:

Chrome

  1. Open chrome://settings/search#certificates
  4. Now, select the root.crt file, and complete the wizard.
  5. Finally, restart Chrome. Done.

Firefox

  1. Open about:preferences#advanced
  4. Now, select the root.crt file, and complete the wizard.
  5. Finally, restart Firefox. Done.

Internet Exploder

  3. Now, select the root.crt file, and complete the wizard.
  4. Uninstall Internet Explorer, and use a real browser. Done.
